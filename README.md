# GradingTool

A comprehensive utility tool for grading student problem submissions and merging them into daily grade sheets.

## Features
- **grade_problem.py**: Python CLI tool to process raw JSON submissions for a single problem, filter by deadline, tie-break, and compute grades.
- **merge_day.py**: Python CLI tool to merge multiple problem sheets, calculate daily averages, support student rosters, and output breakdown details.
- **docs/**: A premium, static client-side web companion app for visual grading, merging, and analytics (ready to host on GitHub Pages).
