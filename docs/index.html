CLI Grading Scripts

Learn how to run the grading engine locally on your machine via Python terminal scripts.

1. Formula details

Individual Task Grade Grade = round( (checks_passed / checks_run) * 5 ) Calculated half-up. Defaults to 0 if checks_run is 0.
Daily Total Degree Total Degree = round( sum(grades) / n ) Where n is the number of problems assigned. Missing submissions score 0.

2. Running Command Line Scripts

If you prefer using the terminal, run the Python scripts available under the GradingTool/ folder. They require zero dependencies and run on standard Python 3.

Script 1: Grader for a Single Problem

python3 grade_problem.py <path_to_raw_submissions.json> --deadline "YYYY-MM-DD HH:MM:SS" --roster roster.csv

Script 2: Merger for a Single Day

python3 merge_day.py ITI_compare_sheet.json ITI_vowels_sheet.json --roster roster.csv --output-prefix ITI_day_3_summary
